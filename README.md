📚 Organizador de Atividades Escolares
## Sobre o projeto
Sistema desenvolvido para ajudar estudantes a organizarem suas atividades escolares. Muitas vezes os alunos esquecem quais tarefas já realizaram e quais ainda estão pendentes — este sistema busca facilitar essa organização de forma simples e prática.
O programa permite cadastrar atividades, acompanhar seu andamento e visualizar rapidamente quais tarefas já foram concluídas e quais ainda precisam ser feitas.
Funcionalidades

## Cadastrar atividades
Listar todas as atividades
Marcar atividades como concluídas
Visualizar atividades pendentes
Visualizar atividades concluídas
Editar e remover atividades
Pesquisar atividades por disciplina

Tecnologias utilizadas

## Python 3
Streamlit (interface gráfica)
JSON (persistência de dados)

Como executar

Instale as dependências: pip install -r requirements.txt
Execute o programa: streamlit run app.py
Acesse no navegador o endereço indicado no terminal (geralmente http://localhost:8501)

Estrutura do projeto

app.py — interface do usuário (Streamlit)
atividades.py — lógica de negócio (cadastrar, editar, filtrar, etc.)
dados.py — persistência dos dados em arquivo JSON
atividades.json — arquivo onde os dados são salvos (gerado automaticamente)
requirements.txt — dependências do projeto
README.md — este arquivo
